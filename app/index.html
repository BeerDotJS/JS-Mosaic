About the Project // Resources

This project was realized in order to take part in the Mozilla Dev Derby a monthly web development contest open to anyone willing to learn.

For the April Dev Derby, web workers was chosen as topic to tackle.

The resorces mainly used were the following:

Web workers
Web workers let you run scripts in the background, making even the most computationally-intensive applications possible.
Canvas
Canvas was used in order to do the image analysis
Coffescript
CoffeeScript is a little language that compiles into JavaScript.
Grunt
A task-based command line tool written in JavaScript.
Instagram
Instagram was used as source for the images
Flask
Flask was used as a proxy to get images from Instagram
